# 🚀 智学秘伴 - 智能学习平台

## 📋 项目概述

"智学秘伴"是一个革命性的智能学习平台，集成了AI技术、3D交互、个性化推荐等先进功能，旨在为学生提供前所未有的学习体验。

## 🌟 核心功能

### 1. 智能学习仪表板
- **实时数据监控**: 专注度、效率、精力、动力等实时指标
- **学习进度追踪**: 多维度学习进度可视化
- **AI智能推荐**: 基于学习历史的个性化内容推荐
- **学习动力系统**: 激励语录、成就系统、目标管理

### 2. 3D智能学习伴侣
- **交互式3D伴侣**: 可互动的3D学习伙伴
- **情感智能**: 根据用户状态调整情绪和行为
- **语音交互**: 支持语音对话和语音合成
- **成长系统**: 亲密度、能量值、解锁功能

### 3. AI学习分析引擎
- **学习模式分析**: 时间分布、专注趋势、效率指标
- **知识缺口识别**: 智能识别学习薄弱环节
- **预测性分析**: 学习成果预测和技能提升预估
- **个性化路径**: AI生成最优学习路径

### 4. 沉浸式学习环境
- **多样化学习空间**: 图书馆、教室、花园、太空等主题环境
- **交互式学习资源**: 可点击的3D学习对象
- **高级视觉效果**: 光影渲染、后期处理、粒子特效

### 5. 智能学习社区
- **社交学习功能**: 学习小组、讨论区、知识分享
- **协作学习**: 实时协作、项目合作、同伴互助
- **成就系统**: 徽章、等级、排行榜

## 🛠️ 技术架构

### 前端技术栈
- **React 18** + **TypeScript**: 现代化前端框架
- **Vite**: 快速构建工具
- **Three.js**: 3D图形渲染
- **Framer Motion**: 流畅动画效果
- **Tailwind CSS**: 现代化CSS框架
- **Lucide React**: 精美图标库

### 后端技术栈
- **Node.js** + **Express**: 服务端运行环境
- **MongoDB**: 文档数据库
- **Redis**: 缓存和会话管理
- **OpenAI API**: AI功能集成
- **WebSocket**: 实时通信

### 部署架构
- **Docker**: 容器化部署
- **Nginx**: 反向代理和负载均衡
- **PM2**: 进程管理
- **监控系统**: 性能监控和日志收集

## 🚀 快速开始

### 环境要求
- Node.js 16+ 
- npm 8+
- 现代浏览器（支持WebGL）

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <项目地址>
   cd 智学秘伴
   ```

2. **安装依赖**
   ```bash
   npm install --force
   ```

3. **启动开发服务器**
   ```bash
   # Windows用户
   start-dev.bat
   
   # 或直接运行
   npm run dev
   ```

4. **访问应用**
   - 打开浏览器访问: http://localhost:5173
   - 默认登录账号: demo / demo123

### 生产环境部署

1. **构建项目**
   ```bash
   npm run build
   ```

2. **使用Docker部署**
   ```bash
   docker-compose up -d
   ```

## 🔧 自定义配置

### 环境变量配置
创建 `.env` 文件：
```env
VITE_API_URL=http://localhost:3000
VITE_OPENAI_API_KEY=your_openai_key
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_KEY=your_supabase_key
```

### 主题定制
修改 `tailwind.config.js` 自定义颜色和样式：
```javascript
theme: {
  extend: {
    colors: {
      primary: '#3b82f6',
      secondary: '#8b5cf6',
    }
  }
}
```
